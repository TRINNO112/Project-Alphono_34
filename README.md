# Project Alphono 34: Anatomy of a Dependent State

## Overview
This is an exclusive, research-based React web application designed to critically examine the functional dependencies, structural vulnerabilities, and external supply lines of Gujarat. It abandons traditional analytical dashboards for a premium "interactive research paper" aesthetic.

## Documentation
- **[Factor Research Guide](factor_research.md):** A structured list of DeepSeek/AI prompts designed to gather specific quantitative data on energy, labor, water, materials, and economics.
- **Project Structure:** Vite + React application, styled with Tailwind CSS, utilizing Framer Motion for scroll-based micro-animations and typography designed for deep reading.

## The Six Pillars of Analysis
1. **Infrastructure & Logistics:** Analyzing maritime gateway concentration, port monopolies, and transport corridor bottlenecks.
2. **Energy Grid & Power Supply:** Assessing the reliance on imported coal and central grid stability.
3. **Industrial Raw Materials:** Tracking the true origins of manufacturing base ingredients.
4. **Water Security:** Evaluating the vulnerability of the Sardar Sarovar single point of failure.
5. **Labor and Demographics:** Understanding the migrant labor ecosystems fueling Surat and Morbi.
6. **Governance & Fiscal:** The impact of diaspora remittances and central subsidization on capital velocity.

## Quick Start
1. Ensure dependencies are installed: `npm install`
2. Start the development server: `npm run dev`
3. View the application locally on `http://localhost:5173`

*(Note: Data visualization components are currently in prototype mode pending final research data injection.)*
